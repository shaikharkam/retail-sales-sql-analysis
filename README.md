# Retail Sales Analysis — SQL Project

## Overview
Analysis of real-world online retail transaction data using MySQL.
The goal was to answer 8 business questions related to revenue, customers, and sales trends.

## Dataset
- Source: UCI Online Retail II Dataset (via Kaggle)
- Period: 2009–2011
- Records: ~1 million transactions

## Business Questions Answered
1. Top 10 products by total revenue
2. Monthly sales trend by year
3. Top 10 customers by total spend
4. Top 10 countries by total revenue
5. Average order value by country
6. Which month had the most orders
7. Countries with most unique customers
8. Total revenue by year

## Tools Used
- MySQL / MySQL Workbench
- Dataset: Excel → CSV import

## Key Skills Demonstrated
- Aggregations (SUM, COUNT, AVG)
- GROUP BY, ORDER BY
- DATE functions (STR_TO_DATE, MONTH, YEAR)
- UNION across multiple tables
- COUNT DISTINCT
- Views
